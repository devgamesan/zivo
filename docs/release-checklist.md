## 共通項目

### 基本操作

- [ ] アプリケーションが正常に起動すること（`zivo` コマンド）
- [ ] ヘルプバーが表示され、主要なキーバインドが確認できること
- [ ] カーソルキー（j/k/↑/↓）でカーソルが移動すること
- [ ] PageUp/PageDown でページ単位の移動ができること
- [ ] Home/End で先頭/末尾に移動できること
- [ ] h/← キーで親ディレクトリに戻れること
- [ ] l/→/Enter キーでディレクトリに入れること
- [ ] Enter キーでファイルを開けること
- [ ] q キーでアプリケーションが終了すること
- [ ] コマンドパレットが開くこと（: キー）
- [ ] Esc キーで選択/フィルターが解除されること

### ファイル操作

#### コピー＆ペースト
- [ ] Space キーでファイル/ディレクトリが選択できること
- [ ] Shift+↑/↓ で範囲選択ができること
- [ ] a キーで表示されている全項目が選択できること
- [ ] c キーで選択項目がコピーされること
- [ ] x キーで選択項目がカットされること
- [ ] v キーでペーストできること
- [ ] ペースト時に名前衝突ダイアログが表示されること
- [ ] 名前衝突時に「上書き/スキップ/リネーム/キャンセル」が選択できること

#### 削除操作
- [ ] d キーでゴミ箱へ移動できること
- [ ] ゴミ箱移動前に確認ダイアログが表示されること（デフォルト）
- [ ] D キーまたは Shift+Delete で完全削除できること
- [ ] 完全削除前に確認ダイアログが表示されること
- [ ] ゴミ箱が空の場合、適切なメッセージが表示されること
- [ ] コマンドパレットから「Empty trash」でゴミ箱を空にできること

#### リネーム＆作成
- [ ] r キーで単一項目のリネームができること
- [ ] n キーでファイルを作成できること
- [ ] N キーでディレクトリを作成できること
- [ ] リネーム/作成時に同名ファイルが存在する場合、適切なエラーが表示されること

#### Undo
- [ ] z キーで直前のファイル操作（リネーム/ペースト/ゴミ箱移動）が取り消せること
- [ ] 複数回 z キーを押すと、操作履歴を順に取り消せること
- [ ] Undo 履歴が空の場合、適切なメッセージが表示されること
- [ ] コマンドパレットから「Undo last file operation」が実行できること

#### アーカイブ操作
- [ ] 単一の .zip/.tar/.tar.gz/.tar.bz2 ファイルを選択し、「Extract archive」で展開できること
- [ ] 展開先の指定ができること
- [ ] 展開先が既存の場合、確認ダイアログが表示されること
- [ ] 複数項目を選択し、「Compress as zip」で圧縮できること
- [ ] 圧縮先の指定ができること
- [ ] 圧縮先が既存の場合、確認ダイアログが表示されること

### プレビュー機能

#### テキストプレビュー
- [ ] テキストファイル（.txt, .py, .md など）を選択すると、右ペインにプレビューが表示されること
- [ ] プレビューがシンタックスハイライトされていること
- [ ] [/] キーでプレビューがスクロールできること
- [ ] 大きなファイルの場合、設定された上限までプレビューが表示されること
- [ ] 設定で `enable_text_preview = false` の場合、プレビューが無効になること

#### 画像プレビュー（chafa がある場合）
- [ ] 画像ファイル（.png, .jpg, .gif など）を選択すると、右ペインにプレビューが表示されること
- [ ] chafa による文字アートが正しく表示されること
- [ ] 設定で `enable_image_preview = false` の場合、プレビューが無効になること
- [ ] chafa がない場合、「install `chafa` for image preview」メッセージが表示されること

#### PDF プレビュー（pdftotext がある場合）
- [ ] PDF ファイルを選択すると、右ペインにテキストプレビューが表示されること
- [ ] 設定で `enable_pdf_preview = false` の場合、プレビューが無効になること
- [ ] pdftotext がない場合、適切なメッセージが表示されること

#### Office ドキュメントプレビュー（pandoc がある場合）
- [ ] .docx/.xlsx/.pptx ファイルを選択すると、右ペインにテキストプレビューが表示されること
- [ ] 設定で `enable_office_preview = false` の場合、プレビューが無効になること
- [ ] pandoc がない場合、適切なメッセージが表示されること

### 検索機能

#### ファイル検索（ripgrep が必要）
- [ ] f キーまたは「Find files」でファイル検索パレットが開くこと
- [ ] 検索文字列を入力すると、結果がリアルタイムにフィルタリングされること
- [ ] 検索結果から Enter キーでファイルにジャンプできること
- [ ] 検索結果でファイルのプレビューが表示されること
- [ ] ripgrep がない場合、適切なエラーメッセージが表示されること

#### grep 検索（ripgrep が必要）
- [ ] g キーまたは「Grep search」で grep 検索パレットが開くこと
- [ ] 検索文字列を入力すると、マッチした行が表示されること
- [ ] ファイル名フィルター（Filename）で結果を絞り込めること
- [ ] 拡張子フィルター（Include/Exclude extensions）で結果を絞り込めること
- [ ] 検索結果から Enter キーでファイルにジャンプできること
- [ ] 検索結果で Ctrl+e キーでエディタを開けること
- [ ] 検索結果でコンテキストプレビューが表示されること

#### 選択ファイルでの grep 検索
- [ ] ファイルを選択状態で「Grep in selected files」が使えること
- [ ] 選択ファイル内のみが検索対象になること

### ナビゲーション機能

#### タブ操作
- [ ] o キーまたは「New tab」で新しいタブが開くこと
- [ ] w キーまたは「Close current tab」でタブが閉じられること
- [ ] 最後のタブは閉じられないこと
- [ ] 1-9, 0 キーで対応するタブに切り替わること
- [ ] Tab/Shift+Tab キーで次/前のタブに切り替わること
- [ ] 各タブで独立したディレクトリを表示できること

#### 履歴
- [ ] H キーまたは「History search」で履歴リストが開くこと
- [ ] { キーまたは「Go back」で前のディレクトリに戻れること
- [ ] } キーまたは「Go forward」で次のディレクトリに進めること
- [ ] 履歴がない場合、適切なメッセージが表示されること

#### ブックマーク
- [ ] b キーまたは「Show bookmarks」でブックマークリストが開くこと
- [ ] B キーまたは「Bookmark this directory」で現在のディレクトリをブックマークできること
- [ ] ブックマーク済みの場合、「Remove bookmark」に変わること
- [ ] ブックマークからディレクトリにジャンプできること
- [ ] config.toml の `[bookmarks].paths` に保存されること

#### パス指定
- [ ] G キーまたは「Go to path」でパス指定ダイアログが開くこと
- [ ] パスを入力すると、補完候補が表示されること
- [ ] Tab キーでパス補完ができること
- [ ] ~ キーまたは「Go to home directory」でホームディレクトリに移動できること
- [ ] 絶対パスと相対パスの両方が使えること

### 転送モード

- [ ] p キーまたは「Toggle transfer mode」で転送モードに切り替わること
- [ ] 2ペインレイアウトが表示されること
- [ ] [/] キーで左右のペインフォーカスが切り替わること
- [ ] 各ペインで独立してナビゲーションできること
- [ ] y キーでフォーカスペインの項目を反対ペインにコピーできること
- [ ] m キーでフォーカスペインの項目を反対ペインに移動できること
- [ ] c/x/v キーでコピー/カット/ペーストが使えること
- [ ] d キーでフォーカスペインの項目をゴミ箱に移せること
- [ ] p/Esc キーでノーマルモードに戻れること

### 設定機能

- [ ] : キーでコマンドパレットを開き、「Edit config」で設定エディタが開くこと
- [ ] ↑/↓ キーで設定項目間を移動できること
- [ ] ←/→/Enter キーで設定値を変更できること
- [ ] s キーで config.toml に保存できること
- [ ] e キーでエディタで config.toml を開けること
- [ ] r キーでヘルプバーがリセットできること
- [ ] Esc キーで設定エディタを閉じられること
- [ ] テーマ変更が即座にプレビューされること
- [ ] config.toml が次回起動時に読み込まれること

#### 主な設定項目の確認
- [ ] `show_hidden_files` の変更が反映されること
- [ ] `show_directory_sizes` の変更が反映されること
- [ ] `enable_text_preview` の変更が反映されること
- [ ] `enable_image_preview` の変更が反映されること
- [ ] `enable_pdf_preview` の変更が反映されること
- [ ] `enable_office_preview` の変更が反映されること
- [ ] `theme` の変更が反映されること
- [ ] `default_sort_field` の変更が反映されること
- [ ] `confirm_delete` の変更が反映されること
- [ ] `paste_conflict_action` の変更が反映されること

### 外部ツール統合

- [ ] e キーまたは「Open in editor」でターミナルエディタが開くこと
- [ ] 設定の `editor.command` が優先されること
- [ ] 次に `$EDITOR` 環境変数が使われること
- [ ] どちらもない場合、ビルトインのデフォルト（nvim/vim/nano）が使われること
- [ ] t キーでフォアグラウンドターミナルが開くこと
- [ ] シェルを exit すると zivo に戻ること
- [ ] T キーまたは「Open terminal」で別ウィンドウのターミナルが開くこと
- [ ] M キーまたは「Open in file manager」でファイルマネージャーが開くこと
- [ ] ! キーまたは「Run shell command」でシェルコマンドを実行できること
- [ ] シェルコマンドの結果がステータスバーに表示されること

### その他の機能

- [ ] / キーでフィルタモードになり、入力した文字列でフィルタリングできること
- [ ] s キーでソート順が循環すること（名前/変更日/サイズ）
- [ ] R キーまたは「Reload directory」でディレクトリが再読み込みされること
- [ ] . キーまたは「Show/Hide hidden files」で隠しファイルの表示が切り替わること
- [ ] i キーまたは「Show attributes」でファイル属性が表示されること
- [ ] C キーまたは「Copy path」でパスがクリップボードにコピーされること
- [ ] 複数選択時、すべてのパスがコピーされること

#### テキスト置換
- [ ] 「Replace text in selected files」で選択ファイルのテキスト置換ができること
- [ ] 「Replace text in found files」でファイル検索結果のテキスト置換ができること
- [ ] 「Replace text in grep results」で grep 検索結果のテキスト置換ができること
- [ ] 「Grep and replace in selected files」で選択ファイルの grep 検索と置換ができること
- [ ] 置換プレビューが右ペインに表示されること
- [ ] Shift+↑/↓ で置換プレビューをスクロールできること

### エッジケース

- [ ] 大きなディレクトリ（1000+ ファイル）でパフォーマンスが著しく低下しないこと
- [ ] 特殊文字を含むファイル名（スペース、日本語、記号など）が正しく扱われること
- [ ] シンボリックリンクが正しく表示・操作できること
- [ ] アクセス権限がないファイル/ディレクトリで適切なエラーが表示されること
- [ ] 読み取り専用ファイルでのコピー/ペーストが正しく動作すること
- [ ] ネットワークドライブ/リモートファイルシステムで基本的な操作ができること
- [ ] 非常に深いディレクトリ階層でナビゲーションが正しく動作すること
- [ ] 非常に長いファイル名が正しく表示されること
- [ ] 設定ファイルが破損している場合、デフォルト値で起動できること
- [ ] 外部ツールが不足している場合、適切なフォールバック動作をすること

## テスト実施上の注意

1. **外部ツールの確認**: 各プレビュー機能をテストする前に、対応する外部ツール（chafa、pandoc、pdftotext）がインストールされているか確認してください
2. **設定のリセット**: テスト前に config.toml をバックアップし、テスト後は元に戻してください
3. **テストデータ**: 各テストケースで十分な種類のファイル（テキスト、画像、PDF、Office ドキュメントなど）を用意してください
4. **OS バージョン**: 各 OS の複数のバージョンでテストすることが望ましいです
5. **アクセシビリティ**: 特に macOS では、フルディスクアクセスの許可設定に注意してください
